# Sistema de Gestión de Escalamientos

API REST desarrollada en Node.js y MongoDB para administrar tickets de fallas y escalamientos (Actividad didactica 3)

## Requisitos previos
- Node.js (v16 o superior)
- MongoDB (local o Atlas)

## Instalar Dependencias en la carpeta del proyecto
npm install

Contenido del archivo env.
MONGODB_URI=mongodb://localhost:27017/escalamientos_db
PORT=3000

iniciar el servidor
npm start
